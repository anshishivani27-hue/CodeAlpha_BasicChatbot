# 🤖 Basic Chatbot

## 📌 Description
A simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs such as greetings, time requests, and common questions.

## ✨ Features
- Greets the user
- Answers predefined questions
- Displays current system time
- Responds to "thank you"
- Exits when the user types "bye"

## 🛠 Technologies Used
- Python 3

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python chatbot.py
```

## 📷 Sample Output

```
Welcome to AI ChatBot!

You: hello
Bot: Hello! How can I help you?

You: time
Bot: 15:30:45

You: bye
Bot: Goodbye! Have a nice day.
```
